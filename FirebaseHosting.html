Firebase Hosting

Hosting: alojamiento web, es el servicio que provee a los usuarios de internet.

Firebase Hosting te brinda una manera rapida, segura y confiable de alojar los elementos estaticos de tu app (HTML, CSS, JavaScript, archivos multimendia. entre otros), ademas de entregar contenido dinamico y alojar microservicios.

Es un servicio de hosting de contenido web, sus funciones clave son:

  • Entregar contenido mediante una conexion segura.
  • Alojar contenido dinamico y estatico, ademas de microservicios.
  • Publicar contenido con rapidez.
  • implementar versiones nuevas con un comando.
  • realizar la reversion con un click.

Implementacion:

  • Se instala firabase CLI.
  • Se inicializa un proyecto firebase y se selecciona para configurar hosting.
  • Se selecciona un proyecto de firebase para conectar al directorio del proyecto local.
  • Especifica un directorio para usar como directorio raiz publica.
  • Crea una configuracion para tu sitio(firebase se encarga automaticamente de agregar las configuraciones de reescritura).

Ventajas:

  • Nos permite tener varios sitios en el mismo proyecto de firebase.
  • Esta isto para trabajar con todos los demas servicios de firebase como cloud functions, authentications...
  • Nos permite implementaciones en firebase desde varios IDE en la web como Stackblitz y GLitch
